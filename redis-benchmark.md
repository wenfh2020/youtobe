# redis-benchmark

Stress test redis 6.0 multi-threaded processing network i/o

---

## obj

* redis-benchmark
* redis-server
* benchmark.sh
* tmux
* htop

---

## download redis 6.0

```shell
cd ~
cd src/other
wget https://codeload.github.com/antirez/redis/zip/6.0
tar zxf 6.0
```

---

## build

```shell
cd redis-6.0/src
make clean; make
```

---

## script

```shell
cd ..
wget https://raw.githubusercontent.com/wenfh2020/shell/master/redis/benchmark.sh
chmod +x benchmark.sh
```

---

## test

* copy config

```shell
cp redis.conf redis.conf.ths
cp redis.conf redis.conf.unths
vim redis.conf.ths
```

* update redis.conf.ths

```shell
# redis.conf.ths
io-threads 4
io-threads-do-reads yes
```

* command

```shell
ulimit -n 16384
./benchmark.sh
```
